# MonkeySwim — Promotional Assets

Store-listing art and screenshots for MonkeySwim. Modeled on the sibling
*Brick Basher* promo layout (`C:\Users\cleon\breakout\promo`).

All graphics are **procedural** (generated by `_gen_art.py` with PIL) and use the
in-game palette from `app/src/main/res/values/colors.xml`, so the promo art
matches the actual game sprites and brand.

## Layout

```
promo/
  _gen_art.py                       generator for the icons + feature graphic
  store-listing-assets/
    icon.png                        512×512 — primary app icon (the brand face)
    icon_options/                   three icon directions to choose from
      icon_A_face.png               monkey face on water (= primary)
      icon_B_chase.png              monkey + piranhas chasing
      icon_C_porthole.png           monkey peering through a brass porthole
    feature_graphic.png             1024×500 — Google Play feature graphic
    screenshots/                    store-ready phone screenshots (debug bar removed)
  raw-screenshots/                  unedited device captures (still show the debug bar)
```

## App icon

Three directions in `icon_options/`. `icon.png` (= `icon_A_face.png`) is the
recommended primary: it mirrors the existing adaptive launcher icon
(`ic_launcher_foreground.xml`) but with the improved, shaded monkey face.
Google Play store icon spec: **512×512 PNG**.

## Feature graphic

`feature_graphic.png` — **1024×500** (Google Play spec). Bold gold "MONKEY
SWIM" wordmark + tagline on the left, the monkey and all four piranha colors on
the right over a depth-graded water background.

## Regenerating

```sh
cd promo
python _gen_art.py        # rewrites icons + feature_graphic.png
```

Screenshots are recaptured manually via `adb exec-out screencap`; the debug bar
is removed with a fixed-band crop (y≈335–424 on a 1080-wide capture).
